## URL del Proyecto

## Nombre del Equipo
Los backends de Sistemas

## Descripción
Aplicación web multipágina desarrollada para MistiTours, una agencia de turismo especializada en tours por Arequipa y el sur del Perú. El sitio ofrece una experiencia completa con un diseño moderno en tonos azul y naranja que refleja la identidad de la marca. Incluye catálogo interactivo de 8 destinos turísticos con modales informativos, página de transporte con flota completa de vehículos (desde sedanes hasta minibuses), formulario de reservas con validación en tiempo real y capacidad de vehículos, sección "Nosotros" con valores de la empresa, y un chatbot inteligente flotante que responde consultas sobre tours, precios, pagos y horarios. Implementa sistema de navegación entre páginas mediante parámetros URL para abrir modales específicos directamente desde la página principal.

## Lenguajes y Tecnologías Utilizadas
- **HTML**: Estructura semántica de 6 páginas principales (Inicio, Nosotros, Transportes, Destinos, Contacto, 404), incluyendo formularios complejos, grids de tarjetas y sistema de modales.
- **CSS**: Diseño responsive con 7 archivos CSS modulares, implementando Grid y Flexbox para layouts adaptativos, animaciones keyframe personalizadas, efectos glassmorphism y parallax, gradientes lineales en hero sections y sistema de variables CSS para paleta de colores consistente.
- **JavaScript**: Interactividad completa con 3 scripts especializados: validación de formulario en tiempo real con verificación de capacidad de vehículos, sistema de modales dinámicos con apertura por URL, y chatbot inteligente con respuestas contextuales sobre tours, precios y servicios.
- **Python**: Lenguaje del backend manejando la lógica del servidor, procesamiento de formularios POST y manejo de errores HTTP (404 personalizado).
- **Flask**: Framework web de Python para gestión de 6 rutas principales, renderizado de templates Jinja2, manejo de métodos HTTP (GET/POST) en formularios y sistema de manejo de errores personalizado.
